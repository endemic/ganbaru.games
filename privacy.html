Privacy Policy

tl;dr: We don't track users.

Game progress is stored in each player's browser, via the "local storage" mechanism. Loading game files from ganbaru.games will create web server log entries; these contain IP addresses and user agents, and are deleted after 20 days. They are used for diagnostic purposes only, and are not shared with any third parties.
